# E3 · Витрина-канал в Telegram (бесплатный рост)

**Owner:** маркетолог + backend · **Волна:** СЕЙЧАС

## Зачем

Самый дешёвый канал привлечения — сам продукт. Аудитория живёт в Telegram: публичный канал, куда наш бот
постит лучшие сигналы **с задержкой 30–60 мин**, каждый пост = живое доказательство + CTA на подписку.
Позиционирование везде: **«Узнай за 40 минут до того, как это станет мейнстримом»**.

## Задачи (defined)

| Task | Что | DoD кратко |
|------|-----|------------|
| TASK-044 · Showcase авто-постинг | Beat-задача: топ-сигналы showcase-тенанта (из TASK-039) старше N минут → пост в публичный канал через нашего бота (ops-токен из TASK-035) + CTA-ссылка с utm. Анти-спам: не чаще M постов/день, score-порог — константы | канал наполняется сам; каждый пост содержит время обнаружения + CTA |
| TASK-045 · Proof-of-speed + авто-кейсы | В пост добавляется «обнаружено в 14:02»; сигналы score>90 складываются в `showcase_cases` (тема, время, каналы) — сырьё для лендинга/рекламы. Сравнение с мейнстримом — пока полуручное (поле «mainstream_at» проставляется оператором) | копится база кейсов «мы раньше мейнстрима» |
| TASK-046 · Реферальная программа USDT | Реф-код на юзера, начисление при первой оплате реферала (через billing webhook), выплата вручную (MVP). Touch: users.ref_code, billing webhook, страница «Пригласи» | привёл друга → получил USDT; учёт в БД, выплата руками |

Зависимости: 044 требует TASK-039 (showcase-тенант) и TASK-035 (ops-бот). 046 независим.

## Финальная точка эпика

Публичный канал автоматически наполняется доказательными постами с путём к подписке;
в каждом посте — пруф скорости; реф-программа работает. Метрика: подписчики канала → регистрации (utm).
