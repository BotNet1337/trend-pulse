## 5. Break-even и путь к $2k MRR

- **Break-even (fixed ~$30/мес):** 2 платящих Pro. Порог выживания тривиален —
  узкое место не затраты, а дистрибуция.
- **$2k MRR:** ~55–69 чистых Pro, либо реалистичный микс ~40 Pro + 8 Trader ≈ $1.95k.
- Инфра при $2k MRR ≈ $100–135 → **операционная маржа ~93–95%** (до стоимости времени owner'а).
- Конверсия Free→Paid **(допущение)**: 3–5% (бенчмарк PLG-SaaS). Для 60 платящих нужно
  ~1200–2000 активированных Free-юзеров → главный KPI воронки (E6): регистрация → пак
  подключён → первый алерт → 👍 → оплата.

## 6. Что ломает экономику (чувствительность)

| Риск | Эффект | Защита |
|---|---|---|
| Churn >30%/мес (продление руками) | LTV ~$90, платный маркетинг бессмыслен | E4 (047/048) — год/quarterly, one-click, grace |
| Бан пула TG-аккаунтов | продукт «молчит» → массовый отток | TASK-035 (health/self-alert), TASK-054 (fallback TGStat) |
| Embeddings per-user без кэша | CPU растёт O(юзеры×посты) | закрыто TASK-037; финально — TASK-052 |
| Низкая конверсия Free→Paid (<1%) | нужна воронка ×5 | E2 (точность сигнала), E5 (упаковка ценности) |
| Реферальный фрод | выплаты без выручки | reward < первый платёж; UNIQUE referred_user_id; ручная выплата на MVP |
